### Description

通过 cc-switch 一个面板管理 4 个 AI CLI 应用的 MCP Servers，双向同步配置。

### How It Works

cc-switch 提供统一的 MCP Server 管理界面，配置一次即可同步到 Claude Code / Codex / OpenCode / Gemini CLI。支持 Server 健康检查和日志查看。
