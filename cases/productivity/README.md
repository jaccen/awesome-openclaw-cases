### Description

将 OpenClaw 变为完全本地 CRM 和销售自动化平台（DuckDB、浏览器自动化、多视图 UI）。

### How It Works

基于 DuckDB 存储客户和交易数据，浏览器自动化抓取 LinkedIn/公司网站补充信息，提供多视图 UI（看板、列表、时间线）。支持自动跟进邮件和销售漏斗分析。
