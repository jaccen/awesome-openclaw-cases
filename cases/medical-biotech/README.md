### Description

将医生信件 / 检验结果 / 处方翻译为通俗语言，提升患者健康素养。

### How It Works

patiently-ai skill 接收医学文档（医生信件、实验室报告、处方说明），识别专业术语并替换为通俗解释，生成适合非医学背景读者理解的版本，保留关键信息（剂量、注意事项）。
