### Description

用自然语言描述编辑视频——裁剪、合并、添加音乐/字幕/调色/竖屏裁剪等操作。

### Prerequisites

- OpenClaw CLI
- FFmpeg
- 视频素材

### How It Works

用户用自然语言描述编辑需求（如"把这段视频裁剪到前30秒，加上字幕背景音乐"），Agent 将其转换为 FFmpeg 命令并执行，支持迭代调整。

### Quick Start

1. 准备视频素材
2. 用自然语言描述编辑需求
3. Agent 生成 FFmpeg 命令并执行
4. 预览结果，迭代调整
